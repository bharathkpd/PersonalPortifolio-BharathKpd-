# Kalavalapudi Bharath Kumar - Premium Portfolio

Welcome to the source code of my personal portfolio website!

## 🚀 Overview

This is a high-end, conversion-focused personal portfolio designed to showcase my skills as a Full Stack Developer, UI/UX Designer, and Python Developer. 

The website is built entirely from scratch using **Vanilla HTML, CSS, and JavaScript**, without relying on heavy frameworks like React or Tailwind. It emphasizes clean code, high performance, and an ultra-premium visual aesthetic.

### 🎨 Design Aesthetic
*   **Theme**: Premium White & Orange
*   **Typography**: Poppins (Headings), Inter (Body), Space Mono (Code/Accents)
*   **Key Features**:
    *   Sleek Bento Box grid layouts
    *   Glassmorphism and frosted UI elements (`backdrop-filter`)
    *   3D parallax hover effects for project cards
    *   Fully responsive mobile-first architecture
    *   Custom CSS animations (typing effects, glowing halos, floating buttons)

## 🛠️ Technical Stack
*   **HTML5**: Semantic and accessible structure.
*   **CSS3**: Advanced CSS Grid, Flexbox, native keyframe animations, and custom CSS variables for easy theming.
*   **JavaScript (ES6+)**: Vanilla JS for intersection observers (scroll reveal animations) and 3D mouse tracking calculations.

## 📂 Featured Sections
1.  **Hero**: Dynamic typing animation and an animated profile display.
2.  **Profile Overview (Bento Box)**: Apple-style grid layout summarizing experience, location, and education.
3.  **Technical Arsenal**: Interactive pill badges highlighting core frontend and backend competencies.
4.  **Projects Grid**: 9 distinct projects with direct links to live deployments and customized thumbnail assets.
5.  **Connect**: Direct integrations with email, phone, GitHub, LinkedIn, and a floating WhatsApp widget.

## 🌐 Live Deployment
This site is designed to be hosted seamlessly via **GitHub Pages**.
